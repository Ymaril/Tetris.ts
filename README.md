# Tetris.ts

*A simple Tetris game written in TypeScript with using SVG.*

## Keys
* **Spacebar** : Start game.
* **Up Arrow** : Rotate left.
* **Left Arrow** : Move shape left.
* **Right Arrow** : Move shape right.
* **Down Arrow** : Lower shape.
* **Enter** : Drop shape.
