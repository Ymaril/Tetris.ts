# Tetris.ts

A simple Tetris game written in TypeScript.
Have fun, and good luck!

<hr>

<h3>Keys:</h3>
* 'Spacebar' : Start game.<br>
* 'Up Arrow' : Rotate left.<br>
* 'Left Arrow' : Move shape left.<br>
* 'Right Arrow' : Move shape right.<br>
* 'Down Arrow' : Lower shape.<br>
* 'Enter' : Drop shape.<br>
