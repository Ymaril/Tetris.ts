# Tetris.ts

A simple Tetris game written in TypeScript.
Have fun, and good luck!

Keys:
* 'Spacebar' : Start game.
* 'Up Arrow' : Rotate left.
* 'Left Arrow' : Move shape left.
* 'Right Arrow' : Move shape right.
* 'Down Arrow' : Lower shape.
* 'Enter' : Drop shape.
